# 常见面试题

# 1、什么是Vue.js？它有哪些优点？

Vue是一个渐进式的JavaScript框架，用户构建用户界面，简单易学、性能高、文档丰富、生态系统完备、灵活性高、可复用性强

# 2、Vue.js的生命周期有哪些？请简述每个生命周期的作用。

Vue2：beforeCreate和created是组件化实例前和实例后的钩子函数；beforeMount和mounted是组件挂载前后挂载后的钩子函数；beforeUpdate和updated是组件更新前和更新后的钩子函数；beforeDestory和destoryed是组件销毁前和销毁后的钩子函数。

Vue3：created生命周期换成setup，语法变成onBeforeXXX和onXXX

# 3、请简述Vue.js的双向数据绑定原理。

Vuejs的数据双向绑定原理是通过数据劫持和发布订阅模式来实现的。当数据改变时，Vuejs会自动更新视图，而当视图改变时也会自动更新数据

# 4、Vue.js中指令和组件有什么区别？

指令是Vuejs中用于操作DOM元素的的特殊属性，常用的指令有v-model、v-bind、v-for、v-if等，而组件是Vuejs中用于封装可复用UI组件的方式，可以通过注册组件进行复用

# 5、Vue.js中的数据响应式是如何实现的？

Vue2中是通过Object.defineproperty()方法实现的，Vue3是通过Proxy和Reflect来实现的。当数据发生变化是，自动的更新依赖

# 6、Vue.js中的computed和watch有什么区别？

computed是用于计算属性的特殊方法，他会依赖自动缓存结果；而watch是用来监听数据变化而执行相关的自定义操作的，可以监听到每次数据的变化

# 7、Vue.js中的路由是什么？请简述Vue.js中的路由实现原理。

路由是用于控制页面跳转的机制，Vuejs中通过vue-router实现，其原理是通过监听URL变化，根据不同的URL展示不同的组件

# 8、Vue.js中的vuex是什么？请简述vuex的作用和使用场景。

Vuex是一个状态管理工具，用来管理全局数据和状态。包括：集中管理共享状态、规范化状态变化、方便调试和测试、提高代码的复用率。使用场景包括：多个组件需要共享同一状态、同一状态需要在多个视图中展示、需要对状态进行统一的管理和调试

# 9、Vue.js中如何实现异步组件加载？

异步组件加载可以通过Vuejs的动态导入来实现的。可以使用动态的import语法来实现

# 10、Vue.js中的mixins有什么作用？请举例说明。

mixins是Vuejs中用于多组件间复用代码的方式，它可以将多个组件的公共代码提取出来，形成一个混入对象，然后将其注入到多个组件中，从而实现代码复用
