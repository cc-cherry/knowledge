# 事件与行为

在Apache ECharts的图表中用户的操作将会触发相应的事件。开发者可以监听这些事件，然后通过回调函数做出相应的处理，比如跳转、弹框等

Echarts中的事件对应DOM的事件名称，均为小写字符串。ECharts 支持常规的鼠标事件类型，包括 'click'、 'dblclick'、 'mousedown'、 'mousemove'、 'mouseup'、 'mouseover'、 'mouseout'、 'globalout'、 'contextmenu' 事件 。

myChart.on('click', function(params) {
  // 控制台打印数据的名称
  console.log(params.name);
});

一方面用户可自定义处理流程，另一方面可调用图表提供的函数触发图表组件相关的行为，比如滑动饼图的时候展示一些提示信息等等

# 图表常见的使用场景

  1. 比较数据：适合柱状图、堆积柱状图、直方图、条形图、面积图
  2. 显示趋势：适合折线图、面积图
  3. 分布数据：适合直方图、箱线图、散点图、热力图
  4. 排序数据：适合条形图、表格、热力图
  5. 部分与整体：适合饼图、环形图、堆积柱状图、堆积面积图、100%堆积柱状图
  6. 关联数据：适合散点图、气泡图、线图、热力图
