    # Webpack相关

    # 1、什么是webpack

    webpack是一个用于js的静态模块化打包工具，通过loader转换文件，通过plugin注入钩子

    # 2、loader的基本原理

    loader被翻译为加载器

    webpack本身只能解析js和json文件，为了能够让webpack解析其他形式文件的能力，比如css、image、less、scss、vue等文件

    手写loader:

    loader通常导出一个函数，在配置时可以给函数传参。loader通过配置schema.json配置文件，约束传入的参数。

    导出的函数默认接收webpack传递的匹配的文件内容，而后可对文件内容进行修改返回。

    • 比如在每个文件前面添加署名注释，即可在文件中添加前缀返回即可

    # 3、plugin的基本原理

    plugin给webpack提供扩展，让webpack更灵活。在webpack运行过程中会触发一系列的钩子函数，plugin可以监听这些事件，从而在特定的时间改变webpack输出的结果。
