# 二、具体配置

# 主题配置

这里推荐使用社区版 material theme： Community Material Theme Palenight High Contrast

# 字体配置

这里推荐专用编程字体 mononoki 的斜体(bold)，配合vscode的字体加粗，展现一绝。敲代码体验极佳！！！

# 二、终端配置

# 终端里的颜值担当

微软新出的Windows Terminal，不过好win10的版本在1909版本及以上才可。老电脑属实瑟瑟发抖。
